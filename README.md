## Extra Features

- Colored "A, B, C, D" - checkboxes (Useful for prioritizing tasks)
- Custom background color settings
- Custom workspace background image

## Alternate Checkboxes

| Syntax   | Description |
| -------- | ----------- |
| `- [ ]`  | to-do       |
| `- [/]`  | incomplete  |
| `- [x]`  | done        |
| `- [-]`  | canceled    |
| `- [>]`  | deferred   |
| `- [<]`  | scheduling  |
| `- [?]`  | question    |
| `- [!]`  | important   |
| `- [A]` | prioritize |
| `- [B]` | prioritize |
| `- [C]` | prioritize |
| `- [D]` | prioritize |
